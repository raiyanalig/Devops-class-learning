# Key Concepts Learned

## 1. What is Maven?
- Maven is a build automation and dependency management tool.
- Mostly used in Java projects.

---

## 2. Why Maven is Used
- Automatically downloads dependencies.
- Simplifies project build process.
- Maintains standard project structure.
